# Assets Folder

This folder should contain app icons and splash screens.

## Missing Files (Optional)

The following files are referenced in app.json but not critical for development:
- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen
- `adaptive-icon.png` - Android adaptive icon
- `favicon.png` - Web favicon

## For Development

The app will work without these files. They're only needed for:
- Building production APK/IPA
- App store submissions
- Better visual appearance

## To Add Icons

1. Create or download app icons
2. Place them in this folder with the correct names
3. Ensure proper dimensions (see app.json for requirements)
