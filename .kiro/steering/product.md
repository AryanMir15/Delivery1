# Product Overview

## Multi-Category Delivery Platform

A complete multi-platform delivery and shopping system supporting multiple business categories including groceries, pharmacy, electronics, fashion, and more. Features customer ordering, shop management, rider delivery tracking, and admin dashboard capabilities.

## Core Applications

- **Backend API**: GraphQL-based Node.js server handling all business logic
- **Mobile App**: React Native customer app for browsing shops and ordering products
- **Rider App**: React Native delivery driver app with live GPS tracking and navigation
- **Admin Dashboard**: React/Vite web interface for platform management
- **Web Frontend**: React customer web application

## Key Features

- Multi-role authentication (customer, vendor, rider, admin)
- Real-time order tracking with WebSocket subscriptions
- Shop and product catalog management
- Shopping cart and checkout with multiple payment methods
- Review and rating system
- Coupon and promotion management
- Live GPS tracking for deliveries
- Earnings tracking for riders
- Analytics and reporting
- Multi-category support (15+ business types)

## User Roles

- **Customer**: Browse shops, order products, track deliveries
- **Vendor/Shop Owner**: Manage product catalog, process orders, view analytics
- **Rider**: Accept deliveries, navigate routes, track earnings
- **Admin**: Platform oversight, vendor management, system configuration
